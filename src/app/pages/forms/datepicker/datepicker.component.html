Basic Example

Set mwlFlatpickr attribute.

DateTime

Set mwlFlatpickr [flatpickrOptions]="{'enableTime':true,'dateFormat':'Y-m-d H:i'}" attribute.

Human-Friendly Dates

Set mwlFlatpickr [flatpickrOptions]="{'altInput':true}" attribute.

MinDate and MaxDate

Set mwlFlatpickr [flatpickrOptions]="{'altInput':true,'minDate':'25-01-2023','maxDate':'29-01-2023'}" attribute.

Default Date

Set [flatpickrOptions]="{'altInput':true,'dateFormat':'d M, Y'}" attribute.

Selecting Multiple Dates

Set mwlFlatpickr [flatpickrOptions]="{'altInput':true,'mode':'multiple'}" attribute.

Range

Set [flatpickrOptions]="{'altInput':true,'mode':'range'}" attribute.

Inline

Set mwlFlatpickr [flatpickrOptions]="{'altInput':true,'inline':true}" attribute.
